@keyframes (nötig):

Mit @keyframes lassen sich verschiedene Zustände der Animation definieren.
※ mehrere @keyframes-Selektoren in einer Animation ist erlaubt.

@keyframes animationname
{keyframes-selector {css-styles;}}

mögliche Angaben:
0% bis 100%
from(0%) und to(100%)

Beispiel-1
prozentuale Werte
from und to

animation-name (nötig):

Ein selbst vergebener Name der @keyframes-Animation.

Defaultwert: none

Bitte sehen Sie alle anderen Beispiele an.

animation-duration (nötig):

Dauer der Animation.

Defaultwert: 0

mögliche Angaben:
Sekunden(s) oder Millisekunden(ms)

Bitte sehen Sie alle anderen Beispiele an.

animation-timing-function:

Geschwindigkeitskurve der Animation.

Defaultwert: ease

mögliche Angaben:
linear = cubic-bezier (0,0,1,1),
ease = cubic-bezier (0.25,0.1,0.25,1),
ease-in = cubic-bezier (0.42,0,1,1),
ease-out = cubic-bezier (0,0,0.58,1),
ease-in-out = cubic-bezier (0.42,0,0.58,1),
cubic-bezier (n,n,n,n) [n: zwischen 0 bis 1]

Beispiel-2
linear
ease
ease-in
ease-out
ease-in-out

animation-delay:

Verzögerung der Animation.

Defaultwert: 0

mögliche Angaben:
Sekunden(s) oder Millisekunden(ms)

Beispiel-3
2s

Bitte klicken und 2 Sekunden warten.

animation-iteration-count:

Die Anzahl Wiederholungen der Animation.

Defaultwert: 1

mögliche Angaben:
numerische Zahl,
infinite

Beispiel-4
dreimal
unendlich

animation-direction:

Abspielrichtung einer Animation.

Defaultwert: normal

mögliche Angaben:
normal,
reverse,
alternate,
alternate-reverse

Beispiel-5
normal
reverse
alternate
alternate-reverse

animation-fill-mode:

Endzustand der Animation.

Defaultwert: none

mögliche Angaben:
none,
forwards,
backwards,
both

Beispiel-6
none
forwards
backwards
both

animation-play-state:

Ob die Animation läuft oder pausiert.

Defaultwert: running

mögliche Angaben:
paused,
running

Beispiel-7
paused
running

animation:

Die Kurzschreibweise für die acht Eigen-
schaften animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count, animation-direction, animation-fill-mode, animation-play-state.

animation:
[name] [duration] [timing-function] [delay] [iteration-count] [direction] [fill-mode]
[play-state];
Defaultwert:
none 0 ease 0 1 normal none running

※ die Angaben von animation-name und animation-duration sind nötig.
