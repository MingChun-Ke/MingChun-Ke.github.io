Syntax der Funktion (ohne Parameter):

Beispiel
    //JavaScript:
    function myFirstFun()
    {

Syntax der Funktion (mit Parameter):

Beispiel
    function myFirstFun(x,y,z) {
    return x * y * z;
    }
    document.write(myFirstFun(2,4,6)); // ergibt 48
    //x,y,z sind Parameter.

Funktionen als Variablen verwendet werden:

Beispiel
    var myFun = function (x,y,z) {return x * y * z};
    document.write(myFun(2,4,6)); // ergibt 48
    /*diese Funktion ist auch eine anonyme Funktion
    (Funktion ohne Name).*/
