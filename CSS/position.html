Statische Position (Defaultwert):

Bei statische Position wirkt Eigenschaften
top, right, bottom und left nicht.

Beispiel-1
div-1 position:
static
div-2 position:
static

Eltern-Container
position: static

Relative Position:

Relative Positionierung zur Standardposition.

Beispiel-2

Angaben für <div-1> − top:15px, left:30px

div-1 position:
relative
div-2 position:
static

Eltern-Container
position: static

Absolute Position:

Sitzt an einer absoluten Position.

Beispiel-3

Angaben für <div-1> − top:16px, left:55px

div-1 position:
absolute
div-2 position:
static
originale
div2-
Position

Eltern-Container
position: relative

Beim Scrollen auch fest positioniert.
