Variablen & Arrays:

Variablen

Variablen sind Container für Datenwerte, und die werden durch die
var-Anweisung definiert.

Beispiel
    var a = 4;
    var b = 8;
    var x = a + b;
    document.write(x); //ergibt 12
    //a, b, x sind Variablen.
Beispiel
    var dogName = "Lucy";
    var dogAge = "3"; 
    document.write
    ("Hundename: " + dogName + "<br>" + "Alter: " + dogAge);
    /*Hundename: Lucy
      Alter: 3*/
    //dogName, dogAge sind Variablen.

Die Regeln für selbstvergebene Namen:

  1. die Namen dürfen keine Leerzeichen enthalten.
  2. die Namen dürfen nur aus Buchstaben und Ziffern bestehen.
  3. das erste Zeichen der Namen muss ein Buchstabe sein.
  4. die Namen dürfen keine deutschen Umlaute oder scharfes S enthalten.
  5. die Namen dürfen als einziges Sonderzeichen den Unterstrich "_" enthalten.
  6. die Namen dürfen nicht mit einem reservierten Wort identisch sein.
  7. Variablen sind case-sensitive.
  8. lowerCamelCase.

Zeichenkettenvereinigung mit Zahlen:

Beispiel
    var text1 = "Hallo" + "Welt";
    var text2 = "Hallo" + 4 + 8;
    var text3 = 4 + 8 + "Hallo";
    document.write(text1); //ergibt HalloWelt
    document.write(text2); //ergibt Hallo48
    document.write(text3); //ergibt 12Hallo
    /*Unterschiedliche Reihenfolge ergibt 
    unterschiedliches Ergebnis!*/

Arrays

Array ist ein Datentyp, der mehrere Werte unter einem gemeinsamen
Namen speichert.

Beispiel
    var dogNames = ["Lucy", "Coco", "Jack"];
    document.write(dogNames); //ergibt Lucy,Coco,Jack

Zugriff auf einfache Arrays:

Beispiel
    var dogNames = ["Lucy", "Coco", "Jack"];
    document.write(dogNames[0]); //ergibt Lucy
    document.write(dogNames[1]); //ergibt Coco
    document.write(dogNames[2]); //ergibt Jack
    //Das erste Array-Element beginnt ab [0].

Eigenschaft length:

Die length-Eigenschaft gibt die Anzahl der Felder eines Arrays an.

Beispiel
    var dogNames = ["Amy", "Lucy", "Coco", "Jack", "Bella"];
    document.write(dogNames.length); //ergibt 5

Methoden push() und pop():

Die push-Methode hängt ein oder mehrere Elemente an das Ende eines Arrays an.
Die pop-Methode entfernt das letzte Element aus einem Array.

Beispiel
    var dogNames = ["Amy", "Lucy", "Coco"];
    dogNames.push("Jack", "Bella");
    document.write(dogNames); //ergibt Amy,Lucy,Coco,Jack,Bella

    var dogNames = ["Amy", "Lucy", "Coco", "Jack", "Bella"];
    dogNames.pop();
    document.write(dogNames); //ergibt Amy,Lucy,Coco,Jack

※ Hinweis:

Array-Elemente können sogar Objekte oder andere Arrays sein.

z.B. –
myArray[0] = Date.now;
myArray[1] = myFunction;
myArray[2] = dogNames;
