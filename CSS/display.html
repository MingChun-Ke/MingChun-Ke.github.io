Primären Display-Werte:

inline • block • inline-block • none


         /*------Beispiel-1:------*/
           
/*CSS*/
.container1  { height: 200px;
               width: 300px;
               border: 1px dashed black; }
                          
.container1 p { background-color: #8ACB88; 
                font-size: 14px; }
                          
.container1 .inline { display: inline; }
        
    <!--HTML-->
    <div class="container1">
      <p>Ich bin Default-display:block.</p>
      <p>Ich bin Default-display:block.</p>
      <p>Ich bin Default-display:block.</p>
      <br>
      <p class="inline">Ich bin auf &quot;
      display:inline&quot; gesetzt.</p>
      <p class="inline">Ich bin auf &quot;
      display:inline&quot; gesetzt.</p>
      <p class="inline">Ich bin auf &quot;
      display:inline&quot; gesetzt.</p>
    </div>

display:inline

Verwandelt ein Block-Element in ein Inline-Element.

Beispiel-1

Ich bin Default-display:block.

Ich bin Default-display:block.

Ich bin Default-display:block.


Ich bin auf "display:inline" gesetzt.

Ich bin auf "display:inline" gesetzt.

Ich bin auf "display:inline" gesetzt.

display:block

Verwandelt ein Inline-Element in ein Block-Element.

Beispiel-2
Ich bin Default-display:inline. Ich bin Default-display:inline. Ich bin Default-display:inline.


Ich bin auf "display:block" gesetzt. Ich bin auf "display:block" gesetzt. Ich bin auf "display:block" gesetzt.

display:inline-block

Erzeugt eine Kombination aus Inline- und
Block-Boxen.

Beispiel-3
Default- display: block
Default- display: block
Default- display: block

display: inline- block
display: inline- block
display: inline- block

display:none

Erzeugt kein Element (behält keine Auswirkungen).

Beispiel-4

1. Sie können mich sehen

2. Sie können mich sehen

3. Sie können mich nicht sehen

4. Sie können mich sehen

5. Sie können mich sehen

※ Die Nr.3 wird auf "display:none" gesetzt.

Hinweis:

Typische Block-Elemente -
<div>, <h1> - <h6>, <p>, <form>, <header>, <footer>, <section>...usw.

Typische Inline-Elemente -
<span>, <a>, <img>, <strong>, <em>...usw.
