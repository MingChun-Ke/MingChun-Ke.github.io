transition-property:

CSS-Eigenschaft der Transition.

Defaultwert: all

mögliche Angaben:
Hier gibt eine Liste der veränderbaren Eigenschaften − CSS animated properties

Beispiel-1
height , width

Bitte mit der Maus über den Ball fahren.

transition-duration (nötig):

Dauer der Transition.

Defaultwert: 0

mögliche Angaben:
Sekunden(s) oder Millisekunden(ms)

Beispiel-2
5s

Bitte mit der Maus über den Ball fahren.

transition-timing-function:

Geschwindigkeitskurve der Transition.

Defaultwert: ease

mögliche Angaben:
linear = cubic-bezier (0,0,1,1),
ease = cubic-bezier (0.25,0.1,0.25,1),
ease-in = cubic-bezier (0.42,0,1,1),
ease-out = cubic-bezier (0,0,0.58,1),
ease-in-out = cubic-bezier (0.42,0,0.58,1),
cubic-bezier (n,n,n,n) [n: zwischen 0 bis 1]

Beispiel-3
linear
ease
ease-in
ease-out
ease-in-out

Bitte mit der Maus über die Blöcke fahren.

transition-delay:

Verzögerung der Transition.

Defaultwert: 0

mögliche Angaben:
Sekunden(s) oder Millisekunden(ms)

Beispiel-4
1s

Bitte fahren Sie den Mauszeiger auf den Ball und bleiben 1 Sekunde.

transition:

Die Kurzschreibweise für die vier Eigen-
schaften transition-property, transition-duration, transition-timing-function, transition-delay.

transition:
[property] [duration] [timing-function] [delay]; Defaultwert: all 0 ease 0
